drive-dodge-die

Created a racing game with acceleration, obstacles, and power-ups using Model View Controller (MVC) architecture in python with two other team members. The goal of the game is to get to the checkpoints within the time limit while avoiding obstacles. The farther along the track, the more obstacles there are.

I worked on creating a timer, creating a leader board to save scores, having a scrolling background, and having the cars change speed, distance, and gas level, along with displaying those things on the screen.

To read more about the project, visit our project website: drive-dodge-die
